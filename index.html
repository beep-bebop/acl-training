计划管理

GitHub 数据仓库备份（推荐）

请使用一个单独的私有 GitHub 仓库存训练数据，不要用 Gist 保存隐私训练记录。
代码仓库只负责页面部署；数据仓库只保存计划、训练历史和统计快照。保存配置后，所有本机修改会自动排队备份；Token 只保存在当前设备，不会写入备份 JSON。

AI 辅助

提示：仅保存在本机浏览器 localStorage，用于“模型完善详情”功能。
